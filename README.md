# Dream Video Builder

絵本プロジェクター（Dream Switch 等）向けに  
動画ファイルを **自動で最適な形式・サイズに変換**する  
Windows用アプリです。

---

## できること

- **MP4 / MOV 動画**をまとめて変換
- iPhoneで撮影したMOV動画にも対応
- 絵本プロジェクター向けの解像度に自動調整
- 形式・サイズはすべて自動
- 難しい設定は不要

---

## 対応ファイル形式

### 入力
- MP4（.mp4）
- MOV（.mov）

### 出力
- 絵本プロジェクター用に最適化された動画ファイル  
  （※サイズ・形式は自動調整されます）

---

## 動作環境

- Windows 10 / 11
- Dream Switch（ドリームスイッチ）（旧モデル / 2025年モデル）

---

1. 「詳細情報」をクリック
